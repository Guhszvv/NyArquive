# NyArquive

NyArquive is an online library developed using **Vite and React**, offering an integrated PDF reader.

## Used technologies

- Frontend: React, Vite

- PDF Reader: PDF.js

## How to run

1. `git clone https://github.com/Guhszvv/NyArquive.git && cd NyArquive`
2. `chmod +x ./install.sh`
3. `cd backend && npm run start`
